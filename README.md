## 产出与积分等级

#### 每次释放会消耗释放前算力百分比的CCT
#### 比如当前算力是100 释放的时候消耗的CCT就是100*百分比

|  级别   | 算力  | 产值   | 消耗CCT百分比  |
|  ----  | ----  |----  | ----  | 
| 0  | 0-800 |0.1% | 1%|
| 1  | 801-2000 |0.1%|1%|
| 2  | 2001-12000 |0.2%|2%|
| 3  | 12001-30000 |0.3%|3%|
| 4  | 30001以上 |0.4% |4%​|
